# Mesh Shift Log – Release Notes

## Release: Phase 5B + 5C.1  
Date: 2026-06-25  
Status: Live deployed and sanity checked

## Summary

This release moves Mesh Shift Log further from a local browser-only checklist tool toward a real internal operations app with backend-backed manager workflows.

The biggest changes are backend sync for assets, improved manager visibility, and much better Manager Dashboard navigation.

## Added

### Asset backend sync

- Added Supabase backend support for asset checks.
- Added Supabase backend support for the asset registry.
- Asset checks can now be saved, restored, and reviewed through backend data.
- Asset registry entries can now sync to Supabase when using Email login.
- Staff-code users can still use local mode without backend writes.

### Manager Dashboard asset tools

- Added asset backend status panel.
- Added asset check history/status visibility.
- Added asset registry backend refresh.
- Added backend sync status for asset registry flow.
- Confirmed asset data appears correctly in Supabase.

### Financial/backend continuity

- Existing financial signoff backend remains working.
- Manager history and daily report continue to work with backend data.
- Local fallback behavior remains in place.

### Manager Dashboard navigation

- Added Manager index near the top of the dashboard.
- Added direct jump buttons to key dashboard sections.
- Added automatic scroll-to-top when opening Manager Dashboard.
- Added persistent “↑ Top” button for quickly returning to the top from anywhere on the dashboard.

## Verified

- App loads without white screen.
- Email login works.
- Staff-code login still works.
- Manager Dashboard opens correctly.
- Manager index appears and works.
- “↑ Top” button works.
- Asset registry saves locally and syncs to Supabase.
- Asset registry refresh works.
- Asset check backend status works.
- Build passes.
- Live deploy completed.
- Live sanity check completed successfully.

## Known notes

- Manager Dashboard is still long.
- Next planned UX improvement is collapsible dashboard sections.
- Backend/auth/sync foundation is now strong enough to continue expanding the app safely.
